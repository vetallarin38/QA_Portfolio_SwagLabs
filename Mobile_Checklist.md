#  Mobile Testing Checklist: Telegram App

Цей документ містить перелік специфічних перевірок для мобільної версії додатка Telegram. Він демонструє розуміння особливостей мобільних платформ (iOS/Android), таких як переривання, жести та робота в умовах нестабільної мережі.

##  Характеристики тестового оточення
* **Пристрій:** iPhone 13 / Samsung Galaxy S21
* **ОС:** iOS 17.0 / Android 13
* **Додаток:** Telegram (версія 10.x)

## 1. Interruption Testing (Переривання)
Ці тести перевіряють, як додаток поводиться при зовнішніх втручаннях.

- [ ] **Вхідний дзвінок:** Прийняти дзвінок під час введення повідомлення. Текст має зберегтися в полі введення (Draft).
- [ ] **Push-сповіщення:** Отримання сповіщення від іншого додатка під час активного чату.
- [ ] **Низький заряд:** Поява системного вікна "Battery 20%" не має призводити до збою додатка.
- [ ] **Будильник:** Спрацювання будильника під час запису голосового повідомлення.

## 2. Network Testing (Мережа)
Перевірка стабільності роботи при зміні умов зв'язку.

- [ ] **No Internet:** Спроба надіслати повідомлення без мережі. Має з'явитися іконка годинника (Waiting).
- [ ] **Відновлення зв'язку:** Автоматичне відправлення "завислих" повідомлень після виходу з офлайн-режиму.
- [ ] **Перемикання мереж:** Перехід з Wi-Fi на 4G (і навпаки) під час завантаження відео.
- [ ] **Слабкий сигнал:** Робота інтерфейсу (infinite loaders) при швидкості Edge/2G.

## 3. UI/UX & Usability (Зручність)
Тести на специфічні мобільні особливості інтерфейсу.

- [ ] **Орієнтація екрана:** Перевірка верстки при повороті екрана в альбомний (Landscape) режим.
- [ ] **Правило великого пальця:** Доступність кнопок "Пошук" та "Нове повідомлення" при триманні телефону однією рукою.
- [ ] **Клавіатура:** Поле вводу має автоматично підніматися над системною клавіатурою.
- [ ] **Жести (Gestures):** Спрацювання свайпа вправо для виходу з чату та свайпа вліво для швидкої відповіді.

## 4. Performance & OS Integration (Ресурси та ОС)
- [ ] **Фоновий режим:** Прихід сповіщень (Push), коли додаток згорнутий.
- [ ] **Дозволи (Permissions):** Робота камери всередині чату після надання/відмови у дозволі.
- [ ] **Темна тема:** Коректність контрастності тексту та іконок при зміні системної теми на Dark Mode.
- [ ] **Очищення кешу:** Робота вбудованої функції видалення медіа-файлів з пам'яті телефону.
